About

This is a small demo blog that stores posts & comments locally in your browser. Refreshing the page will keep data (localStorage). Use it as a starting point for a real backend-powered app.

Quick tips

  • Add posts with the "New Post" button.
  • Open a post to read and comment.
  • Data persists using localStorage — to clear, open console and run localStorage.clear().
